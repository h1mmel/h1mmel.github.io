2. （10分）编写完整的程序。输入三个数，输出其中的最大数，最小数。

   > 输入说明：两个整数N1 N2 N3；
   > 
   > 输出说明：最大数 最小数
   > 
   > 输入样例：5 4 9
   > 
   > 输出样例：9 4

   

   ```c
   #include <stdio.h>
   
   int main(int argc, char * argv[])
   {
   	int a = 0, b = 0, c = 0;
   	scanf("%d %d %d", &a, &b, &c);
   	printf("%d %d", (a > b ? a : b) > c ? (a > b ? a : b) : c,  (a < b ? a : b) < c ? (a < b ? a : b) : c);
   	return 0;
   }
   ```
